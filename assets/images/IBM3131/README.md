# Image Assets Folder – IBM3131 Computer Vision (GitBook)

This folder contains image assets used in the GitBook pages for the **IBM3131 Computer Vision** course.

## Purpose

The images stored here support the visual content of the course material, including:

* Diagrams and illustrations of computer vision concepts
* Example inputs and outputs (e.g., image processing results)
* Figures referenced in explanations and tutorials
* Screenshots used in step-by-step guides

## Usage

These images are intended to be embedded within the GitBook pages to enhance understanding and provide visual context. When adding or updating content:

* Reference images using relative paths from this folder
* Ensure filenames are descriptive and consistent
* Optimize images for web use (compressed, appropriate resolution)

## Naming Conventions

To keep the repository organized, please follow these guidelines:

* Use lowercase letters
* Separate words with hyphens (`-`)
* Avoid spaces and special characters
* Example: `edge-detection-example.png`

## Notes

* Only include images that are directly relevant to the course content
* Avoid duplicate or unused files
* Ensure you have the right to use and distribute any added images
